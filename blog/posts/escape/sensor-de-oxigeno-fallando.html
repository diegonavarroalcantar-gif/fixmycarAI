Sensor de oxígeno (O2) fallando – Diagnóstico FixMyCar AI

Tiempo estimado: 10–25 minutos
Dificultad: Fácil–Media

Síntomas de sensor O2 dañado

  • Consumo elevado de combustible
  • Olor fuerte a gasolina
  • Ralentí inestable
  • Aceleración pobre
  • Check Engine: P0130–P0161
  • Código P0420 pero el catalizador está bien

Paso 1 — Revisar lecturas en tiempo real

Sensor delantero (banco 1 sensor 1):

  • Debe oscilar entre 0.1V y 0.9V
  • Si se queda fijo → sensor lento o muerto

Sensor trasero (post-catalizador):

  • Debe ser estable
  • Si oscila igual que el delantero → catalizador ineficiente

Paso 2 — Revisar cableado

Muy común:

  • Arnés quemado por rozar el escape
  • Conector sulfatado
  • Cables abiertos

Paso 3 — Sensor lento

Si responde con retraso → mezcla incorrecta y consumo alto.

Paso 4 — Prueba rápida

  • Aumentar RPM → revisar si la señal oscila rápido
  • Si no cambia → sensor malo
