Cómo interpretar códigos OBD2 – FixMyCar AI

Tiempo estimado: 5–20 minutos
Dificultad: Fácil–Media

¿Qué es un código OBD2?

Es un código generado por la computadora del vehículo cuando detecta una condición anormal en alguno de sus sistemas: motor, transmisión, emisiones, combustible, encendido, etc.

Estructura de un código OBD2

Un código tiene 5 caracteres, por ejemplo: P0301

CarácterSignificado
P (1° letra)Sistema afectado: P = Motor/Transmisión
0 (2° dígito)Tipo: 0 = código genérico, 1 = fabricante
3 (3° dígito)Subcategoría (por ejemplo: encendido)
01 (últimos dos)Falla específica (cilindro 1)

Interpretación rápida por sistema

  • P0xxx → Motor y emisiones (genérico)
  • P1xxx → Específico del fabricante
  • P2xxx → Aire/combustible, emisiones avanzadas
  • P3xxx → Encendido, chispa, misfires

Códigos más comunes por categoría:

  • P0171 / P0174 → Mezcla pobre
  • P0300–P0306 → Misfires
  • P0420 / P0430 → Catalizador ineficiente
  • P0100–P0104 → MAF
  • P0335–P0349 → CKP/CMP

Paso 1 — Verifica si el código es genérico o del fabricante

Si el segundo dígito es:

  • 0 → Genérico (mismo significado en todos los autos)
  • 1 → Específico del fabricante

Ejemplo: P1131 en Ford = sensor O2 con mezcla pobre.
En Toyota puede significar otra cosa.

Paso 2 — Identificar el sistema afectado

PrefijoSignificado
P01xxSensores aire/combustible (MAF, MAP, IAT)
P02xxInyectores / combustible
P03xxEncendido y misfire
P04xxEmisiones / EVAP
P0420/P0430Catalizador
P0500–P0599Marcha mínima, velocidad, etc.

Paso 3 — Relaciona el código con síntomas reales

El MISMO código puede tener 2 o más causas distintas. Por eso siempre se recomienda revisar síntomas además del código.

Ejemplo:

  • P0302 + vibración fuerte en ralentí → Bobina del cilindro 2
  • P0171 + falta de potencia → MAF sucio o fuga de vacío
  • P0420 + olor a azufre → Catalizador obstruido

Paso 4 — No borrar los códigos antes de diagnosticar

Si borras los códigos, eliminas pistas importantes como “pendientes” o “permanentes”.

Paso 5 — Usa nuestras guías avanzadas por subcategoría

  • Códigos P0xxx – Motor y emisiones
  • Códigos P1xxx – Fabricante
  • Códigos P2xxx – Aire/combustible
  • Códigos P3xxx – Misfire y encendido
